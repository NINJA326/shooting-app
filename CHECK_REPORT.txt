NINJA SHOOTING AVERAGE v10.3

【緊急修正】
- v10.2で成長記録・アジリティー記録が消えていたため、v10.1の完成画面をベースに復元。
- JavaScriptが「async is not defined」で途中停止する原因を修正。
- 成長記録タブ・アジリティー記録タブを維持。
- アジリティー種目：シャトルラン／反復横跳び／L字コーンドリル／垂直跳び。
- アジリティー種目はボタン＋プルダウン両方で選択可能。
- localファイルでも保存IDが作れるように修正。
- Apps Scriptは growth_records / agility_records 対応版を同梱。

【アップロード】
テストGitHubにはZIP内の全ファイルをアップしてください。
特に index.html / google_apps_script_code.js / manifest.json / sw.js を差し替えてください。


【v10.4 修正】
- 成長記録の測定日をクリックするとカレンダーで変更できるようにしました。
- 成長記録保存時、選択した測定日で保存されます。
- manifest / sw のバージョンを v10.4 に更新しました。


【v10.5 修正】
- アジリティー記録に4種目の順位一覧を追加。
- 各種目で最新記録の順位を表示。
- 前回順位から今回順位への変動を「前回4位 → 今回2位」の形式で表示。
- L字コーンドリルはタイム種目のため、数値が小さい方を上位として判定。
- Apps Scriptに agilityRankings API を追加。


【v10.6 修正】
- アジリティー順位一覧を、4種目まとめ表示から「種目選択式」に変更。
- 順位が上がった選手は、✨RANK UP表示・明るい背景・強調枠で華々しく表示。
- 順位が下がった場合は、派手な色を使わず通常表示。
- Apps Scriptはv10.5のagilityRankings APIをそのまま利用。
