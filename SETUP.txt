【v12.0 アップロード手順】

1. GitHub の shooting-app にZIP内のファイルをすべて上書きアップロード
2. Commit changes
3. Google Apps Script に google_apps_script_code.js の中身を全部貼り替え
4. デプロイを管理 → 編集 → 新しいバージョン → デプロイ
5. コーチアプリまたは選手アプリを一度開く

初回実行時：
- growth_records の既存データを「身体測定」へ移行
- 成長記録入力 の既存データを「身体測定」へ移行
- agility_records の既存データを「アジリティ測定」へ移行
- アジリティ入力 の既存データを「アジリティ測定」へ移行
- 移行後、上記4つの重複シートを自動削除
- 以後、選手アプリから入力した身長・体重は「身体測定」へ直接保存
- 以後、選手アプリから入力した測定記録は「アジリティ測定」へ直接保存

最終的な記録先：
- 身長・体重：身体測定
- アジリティー：アジリティ測定
- シュート：shooting_records
