【v11.4 LINEリッチメニュー成果表示 アップロード手順】

1. GitHub の shooting-app リポジトリ直下へ、このフォルダ内のファイルをすべて上書きアップロード
2. Commit changes
3. Google Apps Script に google_apps_script_code.js の中身を全部貼り替え
4. デプロイを管理 → 編集 → 新しいバージョン → デプロイ
5. LINEリッチメニューの各ボタンへ下記URLを設定
